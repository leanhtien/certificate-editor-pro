# Certificate Editor Pro

A free browser-based certificate editor for creating, customizing, and batch-exporting professional certificates.

Created by **Jimmy Lee**.

## Overview

Certificate Editor Pro is a lightweight HTML-based tool that helps users design certificates directly in the browser.  
It supports drag-and-drop editing, custom backgrounds, logos, text styling, Vietnamese fonts, decorative lines, batch name export, and PNG image export.

This project is designed to be simple, portable, and easy to use. No installation is required.

## Features

- Drag and drop certificate objects
- Edit text, font size, font family, color, opacity, and layer
- Support for Vietnamese fonts and handwriting-style fonts
- Upload custom background images
- Upload logo, signature, seal, or additional images
- Add decorative lotus lines and ornaments
- Export a single certificate as PNG
- Batch export multiple certificates using a list of names
- Save and load layout configuration as JSON
- Donation QR section for voluntary support

## How to use

1. Open the website.
2. Upload your custom background if needed.
3. Click any object on the certificate to edit it.
4. Change text, position, size, font, color, layer, or opacity.
5. Upload a name list if you want to export certificates in bulk.
6. Click export to download PNG files or ZIP files.
